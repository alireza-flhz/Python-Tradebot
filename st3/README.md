# st3 — MACD + Bollinger (TradingView / Pine Script)

استراتژی سوم پروژه. برخلاف دو استراتژی قبلی که پایتون بودن، این یکی به زبان
**Pine Script v5** نوشته شده تا مستقیم توی **TradingView** اجرا بشه.

## منطق استراتژی

- **MACD** به عنوان تریگر جهت/مومنتوم استفاده می‌شه.
- **Bollinger Bands** برای تأیید موقعیت قیمت (خرید نزدیک باند پایین، فروش نزدیک باند بالا).
- **حد ضرر (SL)** و **حد سود (TP)** به صورت درصدی.

| سیگنال | شرط |
|--------|-----|
| ورود Long  | کراس صعودی MACD **و** بسته‌شدن قیمت زیر/روی باند پایین بولینگر |
| ورود Short | کراس نزولی MACD **و** بسته‌شدن قیمت بالای/روی باند بالای بولینگر |

## نحوهٔ اجرا در TradingView

1. سایت TradingView رو باز کن → تب **Pine Editor** (پایین صفحه).
2. کل محتوای فایل `MacdBollinger.pine` رو کپی و پیست کن.
3. روی **Add to chart** بزن.
4. برای دیدن نتیجهٔ بک‌تست، تب **Strategy Tester** رو باز کن.

## پارامترهای قابل تنظیم

همه از منوی ⚙️ (تنظیمات اندیکاتور) قابل تغییرن:

- **MACD:** Fast (۱۲)، Slow (۲۶)، Signal (۹)
- **Bollinger:** Length (۲۰)، StdDev Mult (۲.۵)
- **Risk:** Stop Loss ٪ (۲.۰)، Take Profit ٪ (۴.۰)
- **Trades:** فعال/غیرفعال کردن معاملات خرید و فروش
