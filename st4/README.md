# st4 — Scalping: Support/Resistance + FVG (Indicator)

اندیکاتور اسکالپ برای تایم‌فریم‌های **1m / 5m / 30m**. فقط روی چارت **رسم می‌کنه**
و سفارش نمی‌ذاره (نسخهٔ Indicator).

## چی روی چارت می‌بینی

| المان | توضیح |
|-------|-------|
| **حمایت/مقاومت (S/R)** | خطوط افقی از روی سقف‌ها و کف‌های سوئینگ (پیوت) |
| **FVG** (Fair Value Gap) | باکس‌های ناحیهٔ عدم تعادل ۳ کندلی (سبز = صعودی، قرمز = نزولی) |
| **فلش BUY / SELL** | سیگنال اسکالپ (موتور EMA + VWAP + RSI + بایاس 30m) |
| **پس‌زمینهٔ کم‌رنگ** | جهت مجاز معامله بر اساس بایاس 30m |

> ⚠️ خطوط **EMA و VWAP دیگه روی چارت کشیده نمی‌شن** (چارت تمیز شد). این مقادیر
> فقط پشت‌صحنه محاسبه می‌شن چون موتور سیگنال ازشون استفاده می‌کنه.

## حمایت و مقاومت چطور تشخیص داده می‌شه
- با **پیوت سوئینگ**: هر سقفی که `srLen` کندل چپ و راستش پایین‌تر باشه → **مقاومت**.
- هر کفی که `srLen` کندل چپ و راستش بالاتر باشه → **حمایت**.
- فقط `srMax` سطح آخر هر طرف نگه داشته می‌شه تا چارت شلوغ نشه.

## FVG چطور تشخیص داده می‌شه
- **FVG صعودی:** کف کندل فعلی بالاتر از سقف دو کندل قبل باشه (گپ پرنشده) → باکس سبز.
- **FVG نزولی:** سقف کندل فعلی پایین‌تر از کف دو کندل قبل باشه → باکس قرمز.

## نحوهٔ اجرا در TradingView
1. **Pine Editor** → کل `ScalpEmaVwapRsi.pine` رو پیست کن → **Add to chart**.
2. چارت رو بذار روی **1m یا 5m**. بایاس 30m خودکار اعمال می‌شه.
3. برای هشدار: راست‌کلیک روی سیگنال → **Add alert** (سیگنال، FVG صعودی و FVG نزولی همه Alert دارن).

## پارامترهای قابل تنظیم (منوی ⚙️)
- **Support/Resistance:** نمایش، Pivot Lookback (۱۰)، تعداد سطوح هر طرف (۵)، نمایش لیبل قیمت
- **FVG:** نمایش، حداکثر تعداد باکس (۱۲)، امتداد باکس به راست (۱۰ کندل)
- **موتور سیگنال:** EMA Fast/Slow، RSI، اجبار VWAP، روشن/خاموش کردن فلش‌ها
- **Colors:** رنگ حمایت، مقاومت، FVG صعودی و نزولی

## نکات معاملاتی
- سطوح S/R محل مناسب برای حد ضرر/سود و انتظار واکنش قیمتن.
- FVGها اغلب به‌عنوان ناحیهٔ برگشت قیمت پر می‌شن؛ ورود نزدیک FVG هم‌جهت با بایاس قوی‌تره.
- فقط در جهت بایاس 30m (پس‌زمینهٔ سبز = Long، قرمز = Short) و در اورلپ لندن-نیویورک معامله کن.
