# st4 — Scalp entry + Price Action

اندیکاتوری که هم **نقطهٔ ورود** رو نشون می‌ده و هم **پرایس اکشن**.

## چی روی چارت می‌بینی

| المان | توضیح |
|-------|-------|
| 🟢 فلش سبز (زیر کندل) | ورود **Long** |
| 🔴 فلش قرمز (بالای کندل) | ورود **Short** |
| خط قرمز افقی | **مقاومت** (از پیوت سوئینگ) |
| خط سبز افقی | **حمایت** (از پیوت سوئینگ) |
| برچسب `BuEng` / `BeEng` | کندل **Engulfing** (پوشا) صعودی / نزولی |
| نقطهٔ سبز زیر کندل | **Hammer / Pin bar** (برگشت صعودی) |
| نقطهٔ قرمز بالای کندل | **Shooting Star** (برگشت نزولی) |

## سیگنال ورود
کراس **EMA 9 و 21** — تند از کند رد شه بالا (سبز) یا پایین (قرمز). ساده و پرتکرار.

## پرایس اکشن
- **حمایت/مقاومت:** سقف/کف سوئینگ (`srLen` کندل در دو طرف). فقط `srMax` سطح آخر هر طرف می‌مونه.
- **الگوهای کندلی:**
  - **Engulfing:** بدنهٔ کندل فعلی، بدنهٔ کندل قبلی رو کامل بپوشونه.
  - **Hammer / Pin (چکش):** سایهٔ پایین بلند (چند برابر بدنه) → رد شدن قیمت‌های پایین → برگشت صعودی.
  - **Shooting Star:** سایهٔ بالا بلند → رد شدن قیمت‌های بالا → برگشت نزولی.

## نحوهٔ اجرا در TradingView
1. **Pine Editor** → کل `ScalpEmaVwapRsi.pine` رو پیست کن → **Add to chart**.
2. روی هر تایم‌فریمی (برای اسکالپ ۱m / ۵m / ۳۰m).
3. Alert برای ورود Long/Short و Engulfing صعودی/نزولی آماده‌ست.

## تنظیمات (منوی ⚙️)
- **Entry signal:** EMA Fast/Slow، فیلتر روند تایم‌بالاتر (اختیاری، پیش‌فرض خاموش)
- **Price Action:** نمایش S/R، Pivot Lookback، تعداد سطوح، نمایش الگوهای کندلی، نسبت سایهٔ Pin/Hammer
- **Colors:** رنگ حمایت و مقاومت

## ترکیب حرفه‌ای (Confluence)
قوی‌ترین ورود وقتیه که چند چیز با هم جور بشن، مثلا:
- فلش سبز **روی یا نزدیک یه حمایت** + یه کندل **Hammer** یا **Bull Engulfing** → ورود Long مطمئن‌تر.
- فلش قرمز **نزدیک یه مقاومت** + **Shooting Star** یا **Bear Engulfing** → ورود Short مطمئن‌تر.

> نکته: خودِ فلش‌ها و الگوها فیلتر همدیگه نیستن؛ همه رو نشون می‌ده تا خودت کانفلوئنس رو ببینی.
