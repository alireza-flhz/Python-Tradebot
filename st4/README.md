# st4 — Scalp entry signal (ساده)

اندیکاتور ساده که فقط **نقطهٔ ورود** رو با فلش نشون می‌ده:

- 🟢 فلش **سبز** زیر کندل → ورود **Long**
- 🔴 فلش **قرمز** بالای کندل → ورود **Short**

همین، بدون چیز اضافه.

## سیگنال چطور صادر می‌شه
بر پایهٔ **کراس EMA**:
- EMA تند (۹) از EMA کند (۲۱) رد بشه **بالا** → فلش سبز (Long)
- EMA تند از EMA کند رد بشه **پایین** → فلش قرمز (Short)

این سیگنال ساده و پرتکراره، پس حتماً روی چارت فلش می‌بینی.

## نحوهٔ اجرا در TradingView
1. **Pine Editor** → کل `ScalpEmaVwapRsi.pine` رو پیست کن → **Add to chart**.
2. روی هر تایم‌فریمی کار می‌کنه (برای اسکالپ ۱m / ۵m / ۳۰m).
3. برای هشدار: راست‌کلیک روی فلش → **Add alert**.

## تنظیمات (منوی ⚙️)
- **EMA Fast / EMA Slow:** طول میانگین‌ها (پیش‌فرض ۹ و ۲۱)
- **Trend filter (اختیاری):** اگه روشنش کنی، فقط در جهت روند تایم بالاتر (۳۰m) سیگنال می‌ده. پیش‌فرض **خاموش** تا سیگنال بیشتری ببینی.
