# st4 — Scalping Signals: EMA Pullback + VWAP + RSI (Indicator)

اندیکاتور اسکالپ برای تایم‌فریم‌های **1m / 5m / 30m**. فقط **سیگنال روی چارت با فلش**
نشون می‌ده و سفارش نمی‌ذاره (نسخهٔ Indicator، نه Strategy).

## منطق چند-تایم‌فریمی (Top-Down)

| تایم | نقش | ابزار |
|------|-----|-------|
| **30m** | بایاس / جهت | EMA 200 (خودکار از 30m خونده می‌شه) |
| **تایم چارت (1m/5m)** | ستاپ | EMA 9 و EMA 21 + VWAP |
| **تایم چارت (1m/5m)** | تأیید | RSI(14) |

### فلش BUY فقط وقتی همهٔ این‌ها هم‌جهت باشن:
- بایاس 30m صعودی (قیمت بالای EMA200 سی‌دقیقه‌ای)
- قیمت بالای EMA21 و بالای VWAP (هم‌جهت با روند)
- کندل فعلی سایه زده پایین تا EMA9 (پولبک) و بسته شده صعودی
- RSI بالای ۵۰

فلش SELL دقیقاً برعکس.

## نحوهٔ اجرا در TradingView

1. **Pine Editor** رو باز کن → کل `ScalpEmaVwapRsi.pine` رو پیست کن → **Add to chart**.
2. چارت رو بذار روی **1m یا 5m**. بایاس 30m خودکار کشیده می‌شه.
3. برای هشدار زنده: روی سیگنال راست‌کلیک → **Add alert** (یا از منوی Alert شرط‌های `st4 Long/Short Signal`).

## نکات معاملاتی (یادت نره)
- فقط در جهت بایاس 30m معامله کن (پس‌زمینهٔ سبز = فقط Long، قرمز = فقط Short).
- بهترین ساعت: **اورلپ لندن-نیویورک** (تقریباً ۱۵:۳۰–۱۹:۳۰ به وقت ایران).
- حد ضرر تنگ پشت آخرین سقف/کف، حد سود حداقل **R:R = ۱.۵ تا ۲**، ریسک هر معامله **۰.۵–۱٪**.

## پارامترهای قابل تنظیم (منوی ⚙️)
- **Bias:** تایم‌فریم بایاس (پیش‌فرض 30m)، طول EMA بایاس (۲۰۰)
- **Entry TF:** EMA Fast (۹)، EMA Slow (۲۱)، EMA Trend (۲۰۰)، RSI (۱۴)
- **Filters:** اجبار هم‌جهتی با VWAP (روشن)
